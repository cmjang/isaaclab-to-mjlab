## What `mjlab-skillkit` is

`mjlab-skillkit` is a focused skill kit for AI coding assistants working with `mjlab`.

It covers two practical workflows:

1. **IsaacLab Migration Skill**  
   Port IsaacLab environments and projects into clean, mjlab-native implementations while preserving task behavior.

2. **mjlab Native Skill**  
   Author new mjlab-native tasks, configs, managers, sensors, RL setup, and mesh workflows directly from local or bundled references.

## Why this release matters

The old branding correctly communicated migration, but underrepresented the project’s second core value: direct mjlab-native authoring.

This release makes the scope explicit and easier to explain, share, and adopt:

- one skill kit
- two core workflows
- mjlab-native output only
