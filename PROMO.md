# mjlab-skillkit Promo Kit

This file contains ready-to-paste positioning copy for launches, repo descriptions, social posts, and internal sharing.

## 中文版

### 一句话定位

`mjlab-skillkit` 是一个面向 AI 编码助手的 mjlab 技能包：既能迁移 IsaacLab 项目，也能直接编写 mjlab-native 任务。

### 仓库简介

`mjlab-skillkit` 是一个为 Codex、Claude Code、Gemini CLI、Cursor、OpenCode 准备的 mjlab 技能包，内置两大核心能力：**IsaacLab 迁移 Skill** 和 **mjlab Native Skill**。它帮助助手把 IsaacLab 项目迁移成 mjlab-native 实现，也能直接基于 mjlab 文档和示例编写新任务、配置、传感器、RL 接线与资产导入流程。

### 核心卖点

- 一个 skillkit，覆盖 **迁移** 和 **原生开发** 两种高价值场景
- 输出目标始终是 **mjlab-native**
- 优先使用本地仓库与 bundled references，减少无效搜索
- 面向真实编码工作流，而不是只做概念解释

### 两大 Skill

#### 1. IsaacLab Migration Skill

- 用于把 IsaacLab 项目迁移到 mjlab
- 强调行为保持：奖励、观测、动作、命令、reset、termination、curriculum
- 提供映射表、迁移规则、坑点、检查清单、复杂任务迁移打法

#### 2. mjlab Native Skill

- 用于直接写 mjlab-native 代码
- 覆盖 task、EnvCfg、scene、manager、sensor、terrain、RL config、mesh import
- 优先复用 mjlab 官方范式和本地示例
